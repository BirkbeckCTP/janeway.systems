Hosting

First and foremost Janeway is an Open Source project. If you want to host and run an install of Janeay yourself you can do so as all our code is available on our Github page.

Why host with us?

As the core developers of Janeway we are uniquely positioned to provide a hosted solution for it, we know it inside out and maintain installations for our own use already.

Press Hosting

With this package we deploy a full instance of Janeway for you to use, so you have access to the press level and you can install as many journals as you like.

This package includes:

  • Fully maintained installation of Janeway
  • 99% uptime hosting
  • Access to Janeway’s Discord channel
  • 100GB of storage
  • Daily backups
  • Access to underlying data via our admin panel
  • We can also spin up a WordPress blog alongside your Janeway install
  • Interested? Get in touch!

Shared Journal Hosting

Shared hosting allows you a single journal within our shared installation of Janeway. Your data is ring fenced to your journal, however. As such there are some limits to this package, but its cheaper for single journals.

  • A journal installed on a fully maintained installation of Janeway
  • 99% uptime hosting
  • 10GB of storage
  • Daily backups
  • Starting from £850 per year, per journal
